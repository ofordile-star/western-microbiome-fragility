This repository contains analysis code, processed data summaries, figures, and supporting materials for the Perspective article:

**Ofordile ON.** *Tipping points in gut ecology: a hidden resilience deficit in industrialised microbiomes.*

The project develops an ecological resilience framework for understanding industrialised gut microbiomes, arguing that reduced genus-level richness and functional redundancy place these systems closer to critical ecological tipping points.

The repository integrates evidence from:
- the IHAT-GUT longitudinal cohort in Gambian children
- astronaut microbiome dynamics during long-duration spaceflight
- ecological resilience and redundancy theory
